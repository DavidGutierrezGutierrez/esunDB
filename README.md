# EsUn API

1. Clonar proyecto

2. instalar paquetes
```
yarn install
```

3. Renombrar archivo
```.env.template``` a ```.env```

4. Cambiar variables de entorno dentro del archivo ```.env```

5. Levantar la Base de Datos
```
docker-compose up -d
```

6. Levantar el proyecto
```
yarn start:dev
```
